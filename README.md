# 💕 今天吃什么/玩什么 — 情侣小助手

让女朋友轻松选择今天想吃的菜和想玩的游戏，选完自动通知你！

## 🚀 使用方法

### 1. 直接打开
双击 `index.html` 文件，用浏览器打开即可使用。

### 2. 配置通知推送（可选）
1. 打开 [PushPlus 官网](https://www.pushplus.plus/) 注册账号
2. 关注 PushPlus 微信公众号
3. 复制你的 Token
4. 在小程序「管理」页面粘贴 Token 并保存

配置完成后，女朋友每次选择都会推送通知到你的微信！

## 📱 功能说明

### 吃什么 🍜
- 显示预设的菜品列表，点击选择
- 支持「随机帮我选」功能

### 玩什么 🎮
- 显示预设的游戏/活动列表，点击选择
- 支持「随机帮我选」功能

### 管理 ⚙️
- 添加/删除菜品和游戏
- 配置推送通知 Token

## 🎨 预设列表

### 菜品
米饭、面条、饺子、披萨、汉堡、寿司、火锅、烧烤、炸鸡、麻辣烫、tacos、便当、蛋糕、奶茶、沙拉

### 游戏/活动
王者荣耀、和平精英、原神、蛋仔派对、Switch、桌游、密室逃脱、剧本杀、看电影、散步、逛街、拍照

## 💡 自定义

你可以随时在「管理」页面添加或删除菜品和游戏，打造专属你们的列表！

## 📝 注意事项

- 数据保存在浏览器本地存储中，清除浏览器数据会丢失
- 如需长期使用，建议部署到服务器
- 推送通知需要网络连接
